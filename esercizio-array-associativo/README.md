# Traccia
Ordinare, togliere i duplicati e convertire in lowercase una lista di stringhe.
Eseguirlo in 2 modi. Il primo con gli strumenti classici di "filtro" Unix con pipe.
Il secondo "a mano" tramite un ciclo (o più cicli annidati) dentro uno script Bash. Considerare l'utilizzo degli array associativi di Bash (vers. >= 4).







## Descrizione
- Primo script con strumenti Unix classici
- Secondo script con array dichiarativo e array associativo
- Terzo script in C# sempre con array associativo
